# Task Manager (Java)

A simple console-based task manager application built using Java.

## Features
- Add new tasks
- Delete tasks
- Update task details
- Set task priority (LOW, MEDIUM, HIGH)
- Set task status (Completed / Not Completed)
- Add task description
- Automatically record creation time

## Concepts Used
- Object-Oriented Programming (OOP)
- ArrayList (Data Structure)
- Enums (Priority & Status)
- Date & Time (LocalDate / LocalDateTime)
- Exception Handling

## How to Run
1. Open the project in any Java IDE (IntelliJ, NetBeans, VS Code)
2. Run `Main.java`
3. Follow the instructions in the console

## Notes
This project was built as a learning step to practice Java fundamentals and OOP design.
